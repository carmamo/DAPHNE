◆ HAL_GPIO_EXTI_Callback()

void HAL_GPIO_EXTI_Callback ( uint16_t  GPIO_Pin)

This function is a callback routine called when an external interrupt is generated by a specified GPIO pin.

Parameters
GPIO_PinSpecifies the pin number for the interrupt line. The function checks if the interrupt was generated by the specified FLIR_VSYNC_Pin and if so, allocates memory from the FramePoolHandle memory pool. If memory allocation is successful, it starts a DMA transfer to receive data from the hspi1 peripheral and disables the EXTI15_10_IRQn interrupt.
